# 布局 (NiVLayout)

NiVLayout 组件用于传统布局页面，包含了 header、footer、default 等插槽。
组件基于 Vuetify 实现，可与 Vuetify 组件配合使用。

更多细节请查看 Vuetify [文档](https://vuetifyjs.com/zh-Hans/features/application-layout/)。
